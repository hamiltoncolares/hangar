# Hangar

Monorepo com frontend (React + Vite + Tailwind) e backend (Fastify + Prisma).

## Estrutura
- apps/web: frontend
- apps/api: backend
- packages/shared: tipos e utilitários compartilhados
- docs: documentação

## Scripts
- npm run dev:web
- npm run dev:api
- npm run build:web
- npm run build:api
